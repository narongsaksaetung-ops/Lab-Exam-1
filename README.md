# Lab-Exam-1
การทดสอบการดึงข้อมูลพาดหัวข่าวทั่วโลกผ่าน GNews API
การทำงานของเว็บไซต์ถูกแบ่งเป็นส่วนๆ ที่เชื่อมโยงกันอย่างเป็นระบบ ดังนี้:

ส่วนเลือกประเทศ (Country Selector):

สิ่งที่ทำได้: มีปุ่มเลือกประเทศทั้งหมด 10 ประเทศ

การเชื่อมโยง: เมื่อผู้ใช้คลิกเลือกประเทศ ระบบจะยังไม่ดึงข้อมูลทันที แต่จะทำการ "จดจำ" รหัสประเทศ (data-country) ไว้ในตัวแปร และแจ้งสถานะให้ผู้ใช้ทราบว่าพร้อมจะโหลดข่าวของประเทศนั้นๆ แล้ว

ปุ่ม "โหลดข่าวล่าสุด" (Main Action):

สิ่งที่ทำได้: เป็นปุ่มหลักที่ใช้กระตุ้นการทำงานตามโจทย์

การเชื่อมโยง: เมื่อกดปุ่มนี้ JavaScript จะนำรหัสประเทศที่จดจำไว้ ไปประกอบเข้ากับ URL เพื่อส่งคำขอไปยัง API Server

การดึงข้อมูลจาก API (Data Source):

แหล่งข้อมูล: ดึงข้อมูลผ่าน GNews API (gnews.io) ซึ่งเป็นแหล่งข้อมูลข่าวระดับโลกที่ให้ข้อมูลที่แม่นยำและมีรูปภาพประกอบที่สมบูรณ์

การประมวลผล: ระบบจะดึงข้อมูลมาทีเดียว 10 ข่าว เพื่อความรวดเร็วในการใช้งาน

ส่วนแสดงผลข่าว (News Display):

การจัดวาง 3-2: ข่าวลำดับที่ 1-3 จะถูกวางเรียงกันในแถวแรก และลำดับที่ 4-5 จะถูกวางขยับมาตรงกลางในแถวที่สอง เพื่อให้เกิดความสมดุล (Balance) ของหน้าจอ

การจัดรูปแบบวันที่: ระบบจะแปลงวันที่จากรูปแบบสากล (ISO) ให้กลายเป็นภาษาไทยที่สมบูรณ์ (เช่น 7 มกราคม 2569) เพื่อให้อ่านง่ายและเหมาะสมกับผู้ใช้ชาวไทย

ปุ่ม "แสดงเนื้อหาข่าวอื่นๆ" (Pagination):

สิ่งที่ทำได้: แสดงข่าวอันดับที่ 6-10 ที่โหลดรอไว้แล้ว

การเชื่อมโยง: เมื่อคลิก ข้อมูลข่าวที่เหลือจะถูกนำมาแปะต่อท้ายทันทีโดยไม่ต้องไปโหลดจาก API ซ้ำอีกครั้ง ช่วยประหยัดการใช้งานเน็ตและ API Quota
